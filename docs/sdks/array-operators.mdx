Array operators allow for checking if an array custom field does or does not contain specific values

### Client SDKs

Array operators will work with all versions of all client SDKs except on device eval client SDKs.

For on device eval sdks below are the supported sdks and minimum versions

js-on-device-eval-client - v3.5.1

### Server SDKs

Array operators require certain SDK versions to work in server SDKs

Node.js - v5.27.0

Node Lite - v0.4.0

Python - v0.42.0

Ruby - v2.1.0

Java/Kotlin Server - v1.26.0

PHP - v3.5.0

Go - v1.31.0

.Net Server - v2.1.0

All of the new [Server Core](https://docs.statsig.com/server-core/) sdks support array operators in all versions
