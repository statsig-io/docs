# Ingesting Cloudflare Logs and Metrics into Statsig

## Overview

This guide walks you through setting up the Cloudflare Logpush worker in your Cloudflare account and configuring it to send logs and metrics to Statsig using the HTTP destination

---

## âœ… Prerequisites

- Cloudflare Paid Plan

## Configuring Logpush Worker

- Go to 'Settings' tab of your worker page
- Enable 'Logpush' option
- Select 'Configure Logpush jobs' option
- Select 'Create a Logpush job'
- Select 'HTTP Destination'
- Enter the URL like following, replace `REDACTED` with your Statsig Server SDK Key, and select 'Continue'
`https://api.statsig.com/v1/log_event/cf_log_drain?header_statsig-api-key=REDACTED&header_Content-Type=application%2Fjson`
- Select the dataset you would like to Statsig to ingest
- Configure the logpush job and select 'Submit'
