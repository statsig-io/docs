Define the application in your `.app.src`.  

```
{applications, [..., statsig]},
```

Set your server secret key as a config for the `statsig` application. In `app.config`, or your [system configuration file](https://www.erlang.org/doc/design_principles/applications.html#configuring-an-application)

```
[{statsig, [statsig_api_key, <<"secret-">>]}].
```

Then, ensure `statsig` starts with:
```erlang
application:ensure_all_started(statsig)
```
