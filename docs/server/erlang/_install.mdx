Add a dependency on statsig to your:

`mix.exs`:
```
{:statsig, "~> 0.0.1"}
```
`rebar.config`:
```
{statsig, "0.0.1"}.
```
`erlang.mk`
```
dep_statsig = hex 0.0.1
```
