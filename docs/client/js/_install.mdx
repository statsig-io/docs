You can install the statsig SDK via npm, yarn or jsdelivr:

<Tabs
  defaultValue="npm"
  values={[
    {label: 'NPM', value: 'npm'},
    {label: 'Yarn', value: 'yarn'},
  ]}>
  <TabItem value="npm">

```shell
npm install statsig-js
```

  </TabItem>
  <TabItem value="yarn">

```shell
yarn add statsig-js
```

  </TabItem>
</Tabs>

**via jsdelivr:**

Statsig is available from [jsdelivr](https://www.jsdelivr.com/package/npm/statsig-js), an open source CDN. We use this installation method for using Statsig on www.statsig.com! Go ahead, inspect the page source.

To access the current primary JavaScript bundle, use:

`https://cdn.jsdelivr.net/npm/statsig-js/dist/statsig-prod-web-sdk.min.js`

To access specific files/versions:

`http://cdn.jsdelivr.net/npm/statsig-js@{version}/{file}`
