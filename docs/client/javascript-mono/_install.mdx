```html
<script src="https://cdn.jsdelivr.net/npm/@statsig/precomputed-evaluations/build/precomputed-evaluations.min.js"></script>
```

Statsig is hosted on the [jsDelivr](https://www.jsdelivr.com/package/npm/@statsig/precomputed-evaluations) CDN.

To access the current primary JavaScript bundle, use:

`https://cdn.jsdelivr.net/npm/@statsig/precomputed-evaluations/build/precomputed-evaluations.min.js`

To access specific files/versions:

`https://cdn.jsdelivr.net/npm/@statsig/precomputed-evaluations@{version}/build/precomputed-evaluations.min.js`
