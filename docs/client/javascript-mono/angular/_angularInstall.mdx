## Installation

You can install the Statsig SDK via npm or yarn:

<Tabs
  defaultValue="npm"
  groupId="js-install-choice"
  values={[
    {label: 'NPM', value: 'npm'},
    {label: 'Yarn', value: 'yarn'},
  ]}>

  <TabItem value="npm">

```shell
npm install @statsig/angular-bindings
```

  </TabItem>
  <TabItem value="yarn">

```shell
yarn add @statsig/angular-bindings
```

  </TabItem>
</Tabs>
