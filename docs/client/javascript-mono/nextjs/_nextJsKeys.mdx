Add your Statsig client key to your `.env.local` file. You can find your key at which you can find at [console.statsig.com/api_keys](https://console.statsig.com/api_keys).

```bash
# .env.local, the NEXT_PUBLIC_ prefix is required for this to be available on the client side
NEXT_PUBLIC_STATSIG_CLIENT_KEY=client-<REPLACE_WITH_YOUR_CLIENT_KEY>
```
