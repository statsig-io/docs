## Installation

Statsig uses a multi-package strategy, so you will need to install both the Statsig client and the Expo
specific bindings.

Documentation for `@statsig/js-client` can be found [here](/client/javascript-sdk).


<Tabs
  defaultValue="npm"
  groupId="js-install-choice"
  values={[
    {label: 'NPM', value: 'npm'},
    {label: 'Yarn', value: 'yarn'},
  ]}>

  <TabItem value="npm">

```shell
npm install @statsig/js-client @statsig/expo-bindings
```

  </TabItem>
  <TabItem value="yarn">

```shell
yarn add @statsig/js-client @statsig/expo-bindings
```

  </TabItem>
</Tabs>
