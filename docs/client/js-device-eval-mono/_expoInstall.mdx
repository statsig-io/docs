## Installation

To add support for Expo, first, add the Statsig JS client outlined [here](/client/javascript-sdk).

Once the Statsig client is installed, you can add the Expo bindings as described below:

<Tabs
  defaultValue="npm"
  groupId="js-install-choice"
  values={[
    {label: 'NPM', value: 'npm'},
    {label: 'Yarn', value: 'yarn'},
  ]}>

  <TabItem value="npm">

```shell
npm install @statsig/expo-bindings
```

  </TabItem>
  <TabItem value="yarn">

```shell
yarn add @statsig/expo-bindings
```

  </TabItem>
</Tabs>
