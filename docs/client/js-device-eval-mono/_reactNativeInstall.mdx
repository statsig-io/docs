## Installation

Statsig uses a multi-package strategy, so you will need to install both the Statsig client and the React Native
specific bindings.

<Tabs
  defaultValue="npm"
  groupId="js-install-choice"
  values={[
    {label: 'NPM', value: 'npm'},
    {label: 'Yarn', value: 'yarn'},
  ]}>

  <TabItem value="npm">

```shell
npm install @statsig/js-client @statsig/react-native-bindings
```

  </TabItem>
  <TabItem value="yarn">

```shell
yarn add @statsig/js-client @statsig/react-native-bindings
```

  </TabItem>
</Tabs>

### Peer Dependecies

The `@statsig/react-native-bindings` package has peer dependencies which will also need to be installed.
[`react-native-mmkv`](https://github.com/mrousavy/react-native-mmkv?tab=readme-ov-file#react-native) is utilized to allow for synchronous storage, similar to LocalStorage in web environments.
