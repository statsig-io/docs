## Installation

Statsig uses a multi-package strategy, so you will need to install both the Statsig client and the React Native
specific bindings.

<Tabs
  defaultValue="npm"
  groupId="js-install-choice"
  values={[
    {label: 'NPM', value: 'npm'},
    {label: 'Yarn', value: 'yarn'},
  ]}>

  <TabItem value="npm">

```shell
npm install @statsig/react-native-bindings-on-device-eval
```

  </TabItem>
  <TabItem value="yarn">

```shell
yarn add @statsig/react-native-bindings-on-device-eval
```

  </TabItem>
</Tabs>
