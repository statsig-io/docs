## Server Core Overview

Server Core SDKs are next-generation SDKs built on a shared Rust core library. They offer:

- **Significantly improved performance**: Up to 5x faster evaluation times
- **Reduced resource usage**: Lower CPU and memory utilization
- **Enhanced features**: Support for Parameter Stores, observability tools, and more
- **Future-proof**: All new features and improvements will only be available in Core SDKs
- **Consistent behavior**: Unified behavior across all SDK languages
