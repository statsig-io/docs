## Creating a StatsigUser

### Legacy Python SDK

```python
from statsig import statsig_user

user = statsig_user.StatsigUser(
    user_id="user_123",
    email="user@example.com",
    ip="192.168.0.1",
    user_agent="Mozilla/5.0",
    country="US",
    custom={
        "subscription_level": "premium"
    }
)
```

### Python Core SDK

```python
from statsig import StatsigUser

user = StatsigUser(
    user_id="user_123",
    email="user@example.com",
    ip="192.168.0.1",
    user_agent="Mozilla/5.0",
    country="US",
    custom={
        "subscription_level": "premium"
    }
)
```
