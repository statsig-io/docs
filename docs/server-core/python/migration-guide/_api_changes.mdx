### Key Package and Class Changes

| Feature | Python Core SDK | Legacy Python SDK |
|---------|---------------|----------------|
| Package | `statsig_python_core` | `statsig` |
| Import | `from statsig_python_core import *` | `from statsig import Statsig` |
| Options | `StatsigOptions` | `StatsigOptions` |
| User | `StatsigUser` | `StatsigUser` |
| Initialize | `statsig.initialize().wait()` | `statsig.initialize()` |
| Check Gate | `statsig.check_gate()` | `statsig.check_gate()` |
| Get Config | `statsig.get_dynamic_config()` | `statsig.get_config()` |
| Get Experiment | `statsig.get_experiment()` | `statsig.get_experiment()` |
| Get Layer | `statsig.get_layer()` | `statsig.get_layer()` |
| Log Event | `statsig.log_event()` | `statsig.log_event()` |
| Shutdown | `statsig.shutdown().wait()` | `statsig.shutdown()` |

For more details on the new API, see [Python Core SDK documentation](/server-core/python-core).
