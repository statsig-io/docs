## What IP addresses will Statsig access data warehouses from?

Statsig currently accesses data warehouses from both the Statsig console service and Statsig data pipelines. If your data warehouse is IP protected, please allowlist the following console and backend IPs based on your geographical location, and notify us that your warehouse is IP protected on Slack.

### Console IPs

- West US: 20.29.232.235
- East US: 172.177.167.120

### Backend IPs

- 20.252.36.2
- 20.190.14.199
