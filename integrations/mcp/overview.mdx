## Statsig MCP Server

The Statsig MCP (Model Context Protocol) server lets tools like Cursor and Claude have similar functionality to the Statsig Console. With this setup, you can ask questions, explore experiments, and access your Statsig data using AI.

To get started quickly, see the main setup page: [Statsig MCP Server](/integrations/mcp).
