## Manual Exposures

Manual exposure logging gives you control over when exposure events are sent to Statsig. This is useful for preventing logging exposures on feature checks that don't result in the user actually experiencing the feature.

### Gates

Check a gate without automatically logging an exposure:

```php
$result = $this->statsig->checkGateWithExposureLoggingDisabled($user, 'a_gate_name');
```

Manually log the gate exposure:

```php
$this->statsig->manuallyLogGateExposure($user, 'a_gate_name');
```

### Dynamic Configs

Get a config without automatically logging an exposure:

```php
$config = $this->statsig->getConfigWithExposureLoggingDisabled($user, 'a_dynamic_config_name');
```

Manually log the config exposure:

```php
$this->statsig->manuallyLogConfigExposure($user, 'a_dynamic_config_name');
```

### Experiments

Get an experiment without automatically logging an exposure:

```php
$experiment = $this->statsig->getExperimentWithExposureLoggingDisabled($user, 'an_experiment_name');
```

Manually log the experiment exposure:

```php
$this->statsig->manuallyLogExperimentExposure($user, 'an_experiment_name');
```

### Layers

Get a layer without automatically logging an exposure:

```php
$layer = $this->statsig->getLayerWithExposureLoggingDisabled($user, 'a_layer_name');
$paramValue = $layer->get('a_param_name', 'fallback_value');
```

Manually log the layer parameter exposure:

```php
$this->statsig->manuallyLogLayerParameterExposure($user, 'a_layer_name', 'a_param_name');
```
