title: Migrating From js-local-eval
description: Learn how to migrate from the legacy js-local-eval SDK to the new on-device evaluation client
---

### Standard Initialize

<CodeGroup>

```typescript js-local-eval (Legacy)
import Statsig from "statsig-js-local-eval";

// initialize returns a promise which always resolves
await Statsig.initializeAsync(
  "client-sdk-key",
  { environment: { tier: "staging" } } // optional, pass options here if needed
);
```

```typescript New
import { StatsigOnDeviceEvalClient } from '@statsig/js-on-device-eval-client';

const client = new StatsigOnDeviceEvalClient(
  'client-sdk-key',
  null,
  { environment: { tier: 'staging' } } // optional, pass options here if needed
);

await client.initializeAsync();
```

</CodeGroup>
