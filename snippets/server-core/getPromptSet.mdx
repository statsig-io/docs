### Getting a PromptSet

A Prompt Set is used during online evaluations to make it easy to run shadow evaluations on your candidate prompts. The live prompt is served directly to the user, while candidate prompts are run in the background for comparison and logging. This function returns the full set of prompt versions—both the live version and all candidate versions—so you can track performance, iterate safely, and gain insights into which prompts work best before making changes live.
